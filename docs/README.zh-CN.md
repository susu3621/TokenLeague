# TokenLeague 文档

TokenLeague 是一个用于跟踪 AI 助手使用量并展示排行榜的 Token 使用排行榜应用。

## 可用文档

- **[Hooks 指南](HOOKS.md)** - 为 Claude Code 和 Codex CLI 配置统计 hooks

## 快速入口

- **接口列表**: `/api` - 查看所有可用 API 端点
- **排行榜**: `/leaderboard` - 查看 Token 使用排名
- **管理后台**: `/admin/users` - 管理用户

## 当前模板已包含

- Flask 应用基础接线
- Session 登录鉴权
- 设置页
- 文档页
- API 列表页
