# TokenLeague 文档

TokenLeague 用来帮助团队记录 AI 助手的 Token 用量与使用效率，并把这些数据用于复盘和持续改进。

## 从这里开始

- **[项目说明（英文）](../README_EN.md)** - 英文首页介绍、部署与功能说明
- **[项目说明（中文）](../README.md)** - 中文首页说明与部署指南
- **[Hooks 指南](HOOKS.md)** - 配置当前对外文档覆盖的数据采集来源

## 快速入口

- **接口列表**: `/api` - 查看所有可用 API 端点
- **排行榜**: `/leaderboard` - 查看 Token 使用排名
- **管理后台**: `/admin/users` - 管理用户

## 当前重点

- 团队级 Token 使用记录
- 通过 Prompt 和时间线指标观察效率
- 安装与部署相关运维文档
